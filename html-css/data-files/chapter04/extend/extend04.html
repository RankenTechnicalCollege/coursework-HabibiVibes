Hexadecimal: Compact/widely supported. But its hard to interpret the color without a tool.

RGB: More intuitive than hex. Easier to visualize the color. But not as compact as hex.

HSL: Most intuitive. Easier to adjust colors (like lightness). But not as widely supported as hex or RGB.

Preferred color code format: rgb(red, green, blue)
Example: rgb(255, 87, 51)
rgba(255, 87, 51, 0.5) (50% transparent)

Reason: RGB is more intuitive than hex and easier to visualize the color. It is also widely supported.

HWB summary:
Hue, Whiteness, Blackness (HWB) is a cylindrical color model that represents colors in terms of their hue, whiteness, and blackness components. It is designed to be more intuitive for humans to understand and manipulate colors compared to other color models like RGB or HSL.
HWB is particularly useful for applications where users need to select or adjust colors based on their perceptual attributes, such as in graphic design or user interface design.

CMYK summary:
CMYK (Cyan, Magenta, Yellow, Key/Black) is a subtractive color model used in color printing. It is based on the principle that colors are created by subtracting light from a white background using inks of different colors. The CMYK model is used in the printing industry because it allows for a wide range of colors to be produced using just four ink colors.
In CMYK, colors are represented as percentages of each of the four ink colors.
