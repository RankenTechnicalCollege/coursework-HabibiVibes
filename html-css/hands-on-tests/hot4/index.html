Smash Burgers

Crispy edges, juicy center - perfection smashed to the grill.

Use 80/20 ground beef, smash hard, and never press after flipping!
  • Beef Blend 80/20
  • Cheese Type Pepper Jack
  • Bun Type Brioche
My Sauce
Mayo, Ketchup, Mustard, Relish, Vinegar, Paprika, Garlic Powder, Onion Powder, Sugar, Salt, Pepper, cayenne.

Why Smash Burgers?
Smash burgers create a delicious crust that locks in juices, delivering maximum flavor in every bite.
